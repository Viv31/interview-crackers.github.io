HTML Interview Questions

1) What is HTML ?
HTML is Markup Language,it's full form is "Hypertext Markup Language".It is used for creating basic structure of a webpage.
2) Why HTML is called "Markup Language" ?
HTML is called "Markup Language" because the code written in Angular Bracket"<>"
3)What is extension of HTML page?
The extension of HTML page is".html,.htm"
