HTML Interview Questions

1) What is HTML ?
HTML is Markup Language,it's full form is "Hypertext Markup Language".It is used for creating basic structure of a webpage.
2) Why HTML is called "Markup Language" ?
HTML is called "Markup Language" because the code written in Angular Bracket"<>"
3)What is extension of HTML page?
The extension of HTML page is".html,.htm"

5) What are the way to define constant in Php?
Constants are name or identifier that can not be changed during execution of the script. There are two ways of defining constants in Php:
  1. Using define() function
  2. Using const() function
6) How to comment in php ?
There are following ways of commenting in php
  • //text goes here
  • # text goes here
  • /**/ text goes here
7) What are the methods of submitting a form in php ?
Methods of submitting form in php are
  1. GET
  2. POST
8) What are diffrent types of loop in php ?
Loops in PHP:
  1. For loop
  2. While Loop
  3. do While Loop
  4. Foreach Loop
9) What is Header() Function in php ?
Header function is used to redirect to a location in php
ex: header("location:index.php");
